1
ติดตั้ง Python และ Libraries
ต้องการ Python 3.9 ขึ้นไป
ℹ️ ตรวจสอบ Python ก่อนด้วยคำสั่ง python --version ต้องได้ 3.9 หรือสูงกว่า

เปิด Terminal / Command Prompt แล้วรันคำสั่งนี้:

# ติดตั้งทุก library ในครั้งเดียว pip install opencv-python easyocr pythainlp pandas
⚠️
การดาวน์โหลด Model ครั้งแรก
EasyOCR จะดาวน์โหลด model ภาษาไทย (~400 MB) อัตโนมัติเมื่อรันครั้งแรก ต้องมีอินเทอร์เน็ตและรอสักครู่
2
จัดโครงสร้างโฟลเดอร์

📁 thai_spell_checker/ ├── thai_spell_checker.py # โปรแกรมหลัก ├── requirements.txt └── 📁 input_files/ ← วางไฟล์ของคุณที่นี่! ├── photo1.jpg ├── banner.png └── video_clip.mp4
✅
ไฟล์ที่รองรับ
รูปภาพ: .jpg .jpeg .png .bmp .webp .tiff
วิดีโอ: .mp4 .mov .avi .mkv .wmv

โฟลเดอร์ input_files/ จะถูกสร้างอัตโนมัติเมื่อ Run ครั้งแรก

3
ปรับการตั้งค่า (ถ้าต้องการ)
แก้ไขที่ส่วน main() ในไฟล์ thai_spell_checker.py
ตัวแปร ค่าเริ่มต้น คำอธิบาย
INPUT_FOLDER "./input_files" โฟลเดอร์ที่วางไฟล์ทั้งหมด
OUTPUT_HTML "./report.html" ชื่อและที่อยู่ของไฟล์รายงาน
SAMPLE_EVERY 1 สุ่มตรวจวิดีโอทุก N วินาที (ยิ่งมากยิ่งเร็ว)
USE_GPU False เปลี่ยนเป็น True ถ้ามี NVIDIA GPU
# ตัวอย่าง: เปลี่ยน path และสุ่มวิดีโอทุก 3 วินาที INPUT_FOLDER = "C:/Users/YourName/Desktop/my_files" OUTPUT_HTML = "C:/Users/YourName/Desktop/report.html" SAMPLE_EVERY = 3 USE_GPU = False

Troubleshooting

❌ ModuleNotFoundError: No module named 'easyocr'

รัน pip install easyocr อีกครั้ง หรือตรวจสอบว่า pip เป็นของ Python ที่ถูกเวอร์ชัน

❌ ไม่พบข้อความในภาพ (ทั้งที่มีตัวอักษร)

ตรวจสอบว่าภาพคมชัด ไม่เบลอ และข้อความมีขนาดใหญ่พอ (แนะนำ >20px) ภาพที่มีพื้นหลังซับซ้อนอาจอ่านได้ยาก

❌ โปรแกรมช้ามากกับวิดีโอยาว

เพิ่มค่า SAMPLE_EVERY เป็น 3–5 วินาที หรือเปิด USE_GPU = True ถ้ามี GPU

❌ คำปกติถูกระบุว่าผิดมากเกินไป

PyThaiNLP ใช้ Dictionary ราชบัณฑิตฯ ซึ่งอาจไม่รวมคำแสลง ชื่อแบรนด์ หรือคำทับศัพท์ เป็นเรื่องปกติ
